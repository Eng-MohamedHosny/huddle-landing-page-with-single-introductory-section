# Frontend Mentor - Huddle landing page with single introductory section

## Overview
This is my solution to the **Huddle landing page with single introductory section** challenge from [Frontend Mentor](https://www.frontendmentor.io).  
The goal of this challenge is to build a Huddle landing page with single introductory section as close as possible to the provided design.

## My Process
1. Planned the layout using semantic HTML elements.
2. Styled the component using CSS, focusing on:
   - Centering the card
   - Styling the image
   - Adding typography, padding, and rounded corners to match the design
3. Tested responsiveness and alignment manually based on the mobile design preview.

- Learned to match design specifications using only CSS (colors, border-radius, fonts, spacing).
